## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ installed
- A Google Cloud Gemni API Key

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/jarvis-ai-agent.git
   cd jarvis-ai-agent
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**
   ```bash
   cp .env.example .env
   # Open .env and add your GEMINI_API_KEY
   ```

4. **Launch JARVIS**
   ```bash
   npm run dev
   ```

   Visit `http://localhost:3000` to interact with JARVIS.
